# AI Notes Workspace

A modern AI-powered Notes Workspace built using the MERN Stack with secure authentication, responsive UI, and intelligent note management features.

---

## 🚀 Live Demo

### Frontend

https://sas-notes-assignment.vercel.app/

### Backend API

https://sas-notes-assignment.onrender.com/

---

## ✨ Features

- Secure JWT Authentication
- Create, Edit & Delete Notes
- AI-powered note assistance using Gemini API
- Responsive modern UI
- Protected Routes
- RESTful API architecture
- MongoDB Atlas integration
- Full-stack deployment

---

## 🛠️ Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

### Backend

- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Gemini API Integration

### Deployment

- Frontend → Vercel
- Backend → Render

# ⚙️ Environment Variables

## Backend `.env`

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
PORT=5000
```

## Frontend `.env`

```env
VITE_API_BASE_URL=http://localhost:5000
```

# 🧑‍💻 Local Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/sumitthakur2103/SAS-Notes-Assignment
cd AI-Notes-Workspace
```

---

## 2️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```bash
http://localhost:5000
```

---

## 3️⃣ Frontend Setup

Open a new terminal:

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```

---

- Start Command:

```bash
npm start
```
