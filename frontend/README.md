# Notes Workspace - Frontend

React + Vite app built with Tailwind. Uses Zustand for simple state and Axios for API calls.

Setup:
1. Copy .env.example to .env and set VITE_API_BASE_URL to your backend (e.g., https://your-render-service)
2. npm install
3. npm run dev

Deploy to Vercel: connect repo, set VITE_API_BASE_URL env var.
